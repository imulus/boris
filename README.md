# Boris

## A Campfire Bot built in Node

### Installation

1. Install Node (if you haven't already) with [Homebrew](http://mxcl.github.com/homebrew/)

        $ brew install node

2. Clone

        $ git clone git@github.com:imulus/boris.git

3. Install Dependencies

        $ npm install -d
  
4. Get it on

        $ bash start.sh
